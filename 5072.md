# Mòdul 5072 – Sistemes d'Aprenentatge Automàtic.

## Unitat 1. Introducció a l'aprenentatge automàtic

L'aprenentatge automàtic (Machine Learning) és una disciplina de la intel·ligència artificial que permet als sistemes aprendre a partir de dades i millorar el seu rendiment sense haver de ser programats explícitament.

Conceptes principals
Dades: informació utilitzada per entrenar els models.
Algorisme: conjunt d'instruccions que permet aprendre patrons.
Model: resultat de l'entrenament d'un algorisme.
Predicció: resposta generada pel model davant de noves dades.
Aplicacions
Recomanació de productes.
Detecció de correu brossa.
Diagnòstic mèdic.
Predicció de vendes.

Unitat 3. Aprenentatge supervisat

És una tècnica en què el model aprèn a partir de dades etiquetades.

Característiques

Cada exemple conté:

Variables d'entrada.
Resultat esperat.
Tipus de problemes
Classificació

Assignar una categoria.

Exemples:

Spam o no spam.
Aprovat o suspés.
Regressió

Predir un valor numèric.

Exemples:

Preu d'una casa.
Consum energètic.
Algorismes més utilitzats
Regressió lineal.
Arbres de decisió.
KNN.
Random Forest.
Unitat 4. Aprenentatge no supervisat

Treballa amb dades sense etiquetes.

Objectius

Identificar estructures i patrons ocults.

Tècniques principals
Agrupament (Clustering)

Agrupa elements similars.

Exemple:

Segmentació de clients.
Reducció de dimensionalitat

Disminueix el nombre de variables mantenint la informació important.

Exemple:

Visualització de grans conjunts de dades.
Algorismes destacats
K-Means.

PCA.
Unitat 5. Avaluació dels models

Permet determinar si el model és capaç de generalitzar correctament.

Mètriques de classificació
Exactitud (Accuracy).
Precisió (Precision).
Sensibilitat (Recall).
F1-Score.
Mètriques de regressió
Error absolut mitjà (MAE).
Error quadràtic mitjà (MSE).
Arrel de l'error quadràtic mitjà (RMSE).
Problemes habituals
Sobreajustament (Overfitting)

El model memoritza les dades d'entrenament.

Infraajustament (Underfitting)

El model no aprén suficientment.

Unitat 7. Aprenentatge profund (Deep Learning)

És una evolució de les xarxes neuronals basada en múltiples capes ocultes.

Característiques
Gran capacitat d'aprenentatge.
Necessita moltes dades.
Requereix elevat poder de càlcul.
Aplicacions
Reconeixement facial.
Traducció automàtica.
Conducció autònoma.
Assistents virtuals.

Esquema resum
